# FoodTrail

FoodTrail is a **modern Restaurant Finder and Rating Application** built using the **MERN (MongoDB, Express, React, Node.js) stack**. This application allows users to search for restaurants, view ratings, and share their own restaurant experiences by submitting reviews.

## Features

- **Search Restaurants**: Find restaurants by name, location, or cuisine.
- **Ratings and Reviews**: Access detailed reviews and ratings for each restaurant.
- **User Review Submission**: Share your dining experiences with others.

## Technology Stack

FoodTrail is powered by the following technologies:

- **MongoDB**: For database operations to manage users, restaurants, and reviews.
- **Express**: Backend server and API creation.
- **React**: Intuitive and responsive front-end application.
- **Node.js**: Server-side execution environment for seamless backend operations.

## Usage

1. Clone this repository:
   ```
   git clone https://github.com/kavinkumar133/FoodTrail.git
   ```

2. Navigate to the project directory and install dependencies:
   ```
   cd FoodTrail
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

Visit `http://localhost:3000` to view the app locally.
